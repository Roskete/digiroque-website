# DIGIROQUE - Landing Page Profesional

Landing page de alto impacto para la marca personal DIGIROQUE, especializada en consultoría de IA, DeFi, Comercio Online y Automatización con Agentes de IA para PYMES.

## Secciones

- **Hero Section** — Título potente, propuesta de valor diferenciadora, CTA persuasivo
- **Beneficios** — 6 beneficios orientados a resultados y transformación
- **¿Por Qué DIGIROQUE?** — Experiencia, profesionalismo, resultados medibles
- **Servicios** — 4 áreas de expertise con descripción clara
- **Testimonios** — 3 testimonios realistas que transmiten confianza
- **Planes** — 3 paquetes comparables (Starter, Professional, Enterprise)
- **FAQ** — 6 preguntas frecuentes que responden objeciones comunes
- **Contacto** — Formulario orientado a conversión + métodos de contacto
- **Footer** — Navegación completa + redes sociales

## Tecnología

- HTML5 semántico
- CSS3 con variables CSS
- JavaScript vanilla (sin dependencias)
- Responsive (mobile-first)
- Optimizado para Lovable / despliegue rápido

## Colores de Marca

- Primario: `#800020` (rojo burdeos)
- Acento: `#c00030`
- Secundario: `#a02040`
- Fondo oscuro: `#0a0012`
- Texto: `#e0e0e0`

## Desarrollo

```bash
# Abrir directamente en navegador
open index.html

# O servir con cualquier servidor local
python3 -m http.server 8000
```
